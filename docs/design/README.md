# Design Archive

Documents in this directory are design context, active design brainstorms, and
implementation history. They are useful for understanding why the system took
its current shape, but they are not automatically canonical for current
behavior.

- [Architecture](architecture.md) is marked `historical-design`.
- [Implementation record](implemented-plan.md) is marked `historical-record`.
- [ngauth brainstorming](ngauth-brainstorming.md) is marked `brainstorm` and is
  not synchronized with code.

Current behavior should be verified against code, tests, and the living docs in
the parent [docs index](../README.md).
